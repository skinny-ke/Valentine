# ❤️ Romantic Valentine Surprise

A fully responsive, interactive web surprise designed to ask that special someone to be your Valentine. 

## ✨ Features
* **Dodging "No" Button:** A playful "No" button that jumps away when the cursor or finger gets close.
* **Automated Email Notifications:** Sends an instant notification when she clicks "Yes."
* **Date Planner:** Includes a second form where she can select a date, time, and leave notes, which are then emailed directly to you.
* **Confetti Celebration:** Uses the `canvas-confetti` library for a magical finish.
* **Fully Responsive:** Optimized for iPhone, Android, and Desktop.
